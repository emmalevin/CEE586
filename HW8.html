Calculate flow using pressure¶

I use the following equation from the class lecture notes to solve for flow L^2/T and for velocity L/T: $$q = \frac{\sqrt{S_0}h^{5/3}}{n}$$

$$v = \frac{\sqrt{S_0}h^{2/3}}{n}$$

# function for flow and velocity
def flow(SKNOT,H,N):
    return (np.sqrt(SKNOT)*H**(5/3))/(N)

Determining stable deltat¶

To determine a stable $\Delta t$, I use the following CFL condition $$\frac{v\Delta x}{\Delta t} \le 1$$ Solving for $\Delta t$ I find the following: $$v \Delta x \le \Delta t $$ So I will pick a $\Delta t$ that is greater than this criteria.

To pick a stable solution I will pick the velocity at the bottom of the hill during the steady state period (800 sec- where it is likely to be highest in all space and time for our simulation). This will yield a proper stable solution.

Using Newton's method to solve for the recession period¶

We have the following function for our recessional period (where x is q from the notes): $$f(x) = x + (x^{2/5})r_e\left(\frac{5}{3}\right)\left(\frac{\sqrt{S_0}}{n} \right)^{3/5}(t-t_r) - r_eL$$

So the derivative is: $$f'(x) = 1 + \frac{2}{5}(x^{-3/5})r_e\left(\frac{5}{3}\right)\left(\frac{\sqrt{S_0}}{n} \right)^{3/5}(t-t_r)$$

And to solve for the solution for x, I use 10 iterations of Newton's Method such that $$x_{k+1} = x_k - \frac{f(h_k)}{f'(h_k)}$$

In [48]:
# functions for Newton's Method
def f(XK, TIME):
    return XK + XK**(2/5)*q*(5/3)*(np.sqrt(S0)/n)**(3/5)*(TIME-tr)-q*L

# function for Newton's Method derivative
def f_prime(XK, TIME):
    return 1 + (2/5)*(XK**(-3/5))*q*(5/3)*(np.sqrt(S0)/n)**(3/5)*(TIME-tr)

# loop through analytical solution
for i in range(len(time)):
    if time[i] <= tc:
        v_array[i] = ((np.sqrt(S0))/(n))*(q*time[i])**(5/3)
    elif time[i] <= tr:
        v_array[i] = ((np.sqrt(S0))/(n))*(q*tc)**(5/3)
    else:
        v_init = v_array[i-1] # set an initial value of x to the previous value in the array
        v_final = v_array[i-1]
        # implement 10 iterations of Newton's Method
        for j in range(9):
            v_final = v_init - f(v_init,time[i])/f_prime(v_init,time[i]) 
            v_init = v_final
        # fill the new value in the array with our converged solution to Newton's Method
        v_array[i] = v_final

Part 4 Analysis¶

Compare, contrast and discuss the approaches used to calculate overland flow in this assignment. What are the advantages and disadvantages of each?

The finite difference method, analytical solution, and ParFlow model all have advantages and disadvantages. First, the ParFlow model allows the user to experiment with the environment of the overland flow scenario. For instance, in this assignment we were able to tinker with the water table depth and hydraulic conductivity. Doing so enables us to draw relationships between the parameters of the environment and overland flow, so we can better understand the system. However, running the ParFlow model is complex and requires us to know very specific details about our system (as you can see above, it differs greatly from the analytical and numerical solutions). The analytical and numerical solutions are simpler to compute and require fewer input parameters than the ParFlow model. However, they do not fully encapsulate all the complexities of the hillslope like the ParFlow model. The analytical solution and numerical solution provide a good baseline for a simple calculation of overland flow, however they are highly dependent on using stable conditions for deltax and deltat. Also, the numerical solution can greatly vary depending on your chosen step sizes.
